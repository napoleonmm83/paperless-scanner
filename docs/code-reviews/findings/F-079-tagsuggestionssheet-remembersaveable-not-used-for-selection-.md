## Problem

`var selectedTagIds by remember(document.id, document.suggestions) { mutableStateOf(document.selectedTagIds) }` re-initializes whenever suggestions change and is not preserved across process death.

## Proposed Fix

Use `rememberSaveable(document.id)` with a Saver, or hoist into the ViewModel. Decide whether suggestions changes should reset selection or preserve user choice.
