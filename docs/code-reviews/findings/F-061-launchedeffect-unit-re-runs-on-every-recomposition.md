# F-061 — LaunchedEffect(Unit) re-runs on every recomposition

**Severity:** `medium` · **Area:** `performance/recomposition` · **Effort:** `XS`

## Problem

`LaunchedEffect(Unit) { ... HSV→Color conversion }` has no meaningful key — it executes on every recomposition. Even if idempotent, it adds work and obscures intent.

## Proposed Fix

Use the actual dependency: `LaunchedEffect(editingLabel?.color) { ... }`. For initialization-only logic prefer `remember(editingLabel?.color) { computeColor() }`.
